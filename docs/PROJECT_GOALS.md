# Project Goals and Purpose

## What is CowCalfTracker2.0

CowCalfTracker2.0 is the second iteration of an app idea I had several years ago to make it easier for small scale livestock operations to better manage their livestock. The first iteration of CowCalfTracker is written using the MERN stack and is still deployed today
at [cowcalftracker.com](https://cowcalftracker.com). 

## Application Goal
The purpose of CowCalfTracker is to provide small scale farmers and ranchers with an afforable, easy to use, solution. I aim to accomplish this via an intuitive modern UI and a well defined extensable API. This software space lacks the well desinged UI piece and it is what will make this application competative. Users must be able to effectively and easily accomplish this goal. 

## Competition
There are a few software application out there currently that aim to accomplish similar things but none are cost effective or aimed at small operations. They are large software suites that often require a respresentative to help you get setup. This is an old school way of thinking and should not be a requirement of CowCalfTracker.
